=== PHPInfo for WP ===
Contributors: netbomber
Tags: phpinfo, helper
Requires at least: 3.0
Tested up to: 3.2.1
Stable tag: 1.0

A simple plugin that displays the phpinfo() function for your site.

== Installation ==

1. Go to the dashboard of your WP install and upload the zip file.
2. Click Install then click Activate.
3. Open the Settings tab, then click on PHP Info.
